# دامەزراندنی دەنگی ژوور (LiveKit) — ٢ تا ٩٩+ یاریزان

هەموو کۆدەکە ئامادەیە. تەنها ٣ هەنگاو ماوە بۆ چالاککردنی دەنگ.
بەبێ ئەم هەنگاوانە، ئەپەکە کاردەکات بەڵام شریتی دەنگ دەرناکەوێت.

---

## ١) دروستکردنی هەژماری LiveKit Cloud (بەخۆڕایی)

1. بڕۆ بۆ 👉 https://cloud.livekit.io
2. **Sign up** بکە (بە Google یان ئیمەیڵ).
3. پرۆژەیەکی نوێ دروست بکە (Create Project) — هەر ناوێک، نموونە `imposter`.
4. دوای دروستبوون، بڕۆ بۆ **Settings → Keys** (یان **API Keys**).
5. سێ شت هەڵبگرە:
   - **WebSocket URL** — وەک `wss://imposter-xxxx.livekit.cloud`
   - **API Key** — وەک `APIxxxxxxxx`
   - **API Secret** — ڕیستی درێژ

> خشتەی بەخۆڕایی: هەزاران خولەکی دەنگ مانگانە — تەواو بەس بۆ دەستپێکردن.

---

## ٢) دانانی نهێنییەکان لە Supabase

لە تێرمیناڵدا (لە فۆڵدەری پرۆژەکە)، یەکەم جار توکنەکە دابنێ، پاشان:

```powershell
$env:SUPABASE_ACCESS_TOKEN="<SUPABASE_ACCESS_TOKEN>"

npx supabase secrets set `
  LIVEKIT_API_KEY=<API_Key> `
  LIVEKIT_API_SECRET=<API_Secret> `
  LIVEKIT_URL=<WebSocket_URL>
```

> `<WebSocket_URL>` دەبێت بە `wss://` دەستپێبکات.

---

## ٣) ناردنی Edge Function

```powershell
npx supabase functions deploy livekit-token
```

> ئەم فەنکشنە بە شێوەی ئاسایی (بە JWT) دەنێردرێت بۆ ئەوەی تەنها بەکارهێنەری
> چوونەژوورەوەکراو بتوانێت توکنی دەنگ وەربگرێت — پاراستن خۆکارە.

---

## ٤) تاقیکردنەوە

1. ئەپ بکەرەوە (`Ctrl + Shift + R`).
2. ژوورێک دروست بکە یان بەشداری ژوورێک بکە.
3. لە خوارەوەی شاشە **شریتی دەنگ** دەردەکەوێت 🎙️.
4. کرتە لە دوگمەی مایک بکە بۆ کردنەوەی → مۆڵەتی مایک بدە.
5. لە ئامێرێکی تر / هەژمارێکی تر بەشداری هەمان ژوور بکە و قسە بکە —
   دەبێت دەنگ ببیستیت و دەوری ناوی قسەکەر بدرەوشێتەوە.

---

## تێبینییەکان

- **٢ تا ٩٩+ یاریزان:** LiveKit وەک SFU کار دەکات — دەنگت تەنها جارێک بۆ
  سێرڤەر دەنێریت و ئەو بۆ هەمووان دەینێرێت. لەبەرئەوە ژووری گەورەش کێشە نییە.
- **تەنها قسەکەرە چالاکەکان:** ئەوانەی بێدەنگن دەنگیان ناگوازرێتەوە، بۆیە
  دەنگ پاک دەمێنێتەوە تەنانەت لەگەڵ ٩٩ کەس.
- **iOS:** دەبێت بەکارهێنەر کرتە بکات بۆ کردنەوەی مایک (سیاسەتی ئۆتۆپلەی Apple).
- **خۆ-ئیستضافە (ئارەزوومەندانە):** دواتر دەتوانیت LiveKit لەسەر سێرڤەری خۆت
  دابمەزرێنیت بەبێ گۆڕینی کۆد — تەنها `LIVEKIT_URL`/`KEY`/`SECRET` بگۆڕە.
