# موسيقى الخلفية (Background music)

ملفّات mp3 هنا تُشغَّل كـ**قائمة تشغيل متواصلة** في الخلفية (تنتقل تلقائيًّا للأغنية التالية).

الأغاني الحالية مرتبطة في `src/lib/sound.js` ضمن `MUSIC_TRACKS`.

## لإضافة أو تغيير أغنية
1. ضَع ملفّ mp3 هنا باسم **إنجليزي بدون مسافات** (مثل `my-song.mp3`).
   - تجنّب المسافات والأقواس والحروف العربية في اسم الملفّ (تسبّب مشاكل في الروابط).
2. أضِف سطرًا في `MUSIC_TRACKS` داخل `src/lib/sound.js`:
   ```js
   { id: 'mysong', name: 'اسم العرض', src: '/music/my-song.mp3' }
   ```
3. `id` فريد، و`name` هو ما يظهر في الإعدادات، و`src` يطابق اسم الملفّ.

## ملاحظات
- إن لم يوجد أيّ ملفّ، يرجع التطبيق تلقائيًّا إلى موسيقى مولّدة (لا تتوقّف أبدًا).
- يُفضّل ملفّات بحجم صغير لتحميل أسرع.
- صوت **بداية اللعبة** (اللقطة الواحدة) في مجلّد منفصل: `public/game-start/start.mp3`.
