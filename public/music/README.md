# موسيقى الخلفية (Background music)

ضَع ملفّات الموسيقى هنا بصيغة **mp3** وبهذه الأسماء بالضبط:

| النمط | اسم الملفّ المطلوب |
|-------|--------------------|
| هادئ  | `calm.mp3`         |
| غموض  | `mystery.mp3`      |
| عربي  | `oud.mp3`          |

ملاحظات:
- إن لم يوجد ملفّ، يرجع التطبيق تلقائيًّا إلى الموسيقى المولّدة (لا تتوقّف الموسيقى أبدًا).
- يُفضّل ملفّات هادئة وحلقية (loop) وحجم صغير (أقل من ~3MB لكل ملفّ) لسرعة التحميل.
- لتغيير الأسماء أو إضافة أنماط جديدة، عدّل `MUSIC_TRACKS` في `src/lib/sound.js`.
- بعد إضافة الملفّات: ارفعها مع المشروع (git) أو ضعها في الاستضافة داخل مجلّد `music/`.
