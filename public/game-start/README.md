# صوت بداية اللعبة (Game start sound)

ضَع هنا ملفًّا واحدًا بصيغة **mp3** باسم:

```
start.mp3
```

كيف يعمل:
- يُشغَّل **مرّة واحدة** عند بدء أيّ لعبة (لحظة دخول مرحلة كشف الأدوار) — لكلّ الأنماط، أونلاين ومحلّي.
- مستقلّ تمامًا عن موسيقى الخلفية في مجلّد `music/` (تلك متواصلة، وهذه لقطة واحدة قصيرة).
- إن لم يوجد الملفّ، يرجع التطبيق تلقائيًّا إلى نغمة مولّدة قصيرة (لا يحدث خطأ).
- يُفضّل أن يكون قصيرًا (2–5 ثوانٍ) وحجمه صغير.

لتغيير الاسم أو مستوى الصوت، عدّل `playGameStart` في `src/lib/sound.js`.
