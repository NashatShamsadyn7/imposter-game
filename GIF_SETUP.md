# دامەزراندنی GIF (GIPHY)

کۆدی GIF ئامادەیە. تەنها کلیلێکی GIPHY (بەخۆڕایی) پێویستە.
ئەگەر کلیل دانەنێیت، چات بەبێ GIF کاردەکات (دوگمەی GIF دەرناکەوێت).

> تێبینی: Tenor (هی Google) بۆ تۆمارکردنی نوێ داخراوە، بۆیە GIPHY بەکاردێنین.

---

## ١) وەرگرتنی کلیلی GIPHY (بەخۆڕایی)

1. بڕۆ بۆ 👉 https://developers.giphy.com/dashboard/
2. هەژمارێک دروست بکە / بچۆ ژوورەوە (بە ئیمەیڵ یان Google).
3. کرتە لە **Create an App** بکە.
4. **API** هەڵبژێرە (نەک SDK) → **Next**.
5. ناوێک و وەسفێک بنووسە → **Create App**.
6. **API Key** ـەکەت دەردەکەوێت — کۆپی بکە.

---

## ٢) دانانی کلیل

**لە Vercel:** Project → Settings → Environment Variables زیاد بکە:

```
VITE_GIPHY_KEY = <کلیلەکەت>
```

**لە `.env` ناوخۆیی** هەمان شت بنووسە، پاشان dev/build دووبارە بکەرەوە.

دوای deployـی نوێ، دوگمەی 🎬 GIF لە چاتی گرووپ و گفتوگۆی تایبەتدا دەردەکەوێت.

---

## ٣) تاقیکردنەوە

1. ئەپ بکەرەوە (`Ctrl + Shift + R`).
2. بڕۆ بۆ گرووپێک یان گفتوگۆی هاوڕێیەک.
3. کرتە لە دوگمەی 🎬 بکە، بگەڕێ، GIF هەڵبژێرە و بینێرە.

> تێبینی: کلیلی بەخۆڕایی (beta) سنووری داواکاریی هەیە؛ بۆ بەرهەمهێنان (production)
> دەتوانیت لە داشبۆردی GIPHY داوای **Production Key** بکەیت.
