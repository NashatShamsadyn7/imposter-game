# دامەزراندنی GIF (Tenor)

کۆدی GIF ئامادەیە. تەنها کلیلێکی Tenor (بەخۆڕایی لە Google) پێویستە.
ئەگەر کلیل دانەنێیت، چات بەبێ GIF کاردەکات (دوگمەی GIF دەرناکەوێت).

---

## ١) وەرگرتنی کلیلی Tenor (بەخۆڕایی)

1. بڕۆ بۆ 👉 https://console.cloud.google.com
2. پرۆژەیەکی نوێ دروست بکە (یان ئەوەی هەیە هەڵبژێرە).
3. لە **APIs & Services → Library** بگەڕێ بۆ **Tenor API** و **Enable** بکە.
4. بڕۆ بۆ **APIs & Services → Credentials → Create Credentials → API key**.
5. کلیلەکە کۆپی بکە.

> ئارەزوومەندانە: کلیلەکە سنووردار بکە بۆ Tenor API + دۆمەینی iosbb.vercel.app

---

## ٢) دانانی کلیل

**لە Vercel:** Project → Settings → Environment Variables زیاد بکە:

```
VITE_TENOR_KEY = <کلیلەکەت>
```

**لە `.env` ناوخۆیی** هەمان شت بنووسە، پاشان dev/build دووبارە بکەرەوە.

دوای deployـی نوێ، دوگمەی 🎬 GIF لە چاتی گرووپ و گفتوگۆی تایبەتدا دەردەکەوێت.

---

## ٣) تاقیکردنەوە

1. ئەپ بکەرەوە (`Ctrl + Shift + R`).
2. بڕۆ بۆ گرووپێک یان گفتوگۆی هاوڕێیەک.
3. کرتە لە دوگمەی 🎬 بکە، بگەڕێ، GIF هەڵبژێرە و بینێرە.
